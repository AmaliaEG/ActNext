Module: GenderPickerInput

GenderPickerInput is another custom React Native component that allows users to select their gender from a predefined list using a modal-based dropdown. The selected gender is then displayed in a touchable field. When pressed, a modal opens showing all gender options. Selecting one updates the selected value and closes the modal.
