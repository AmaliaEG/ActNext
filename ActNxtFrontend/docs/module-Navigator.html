Module: Navigator

Sets up the app's main navigation structure using a Drawer navigator. Wraps the entire navigation in a ThemeProvider and Auth0Provider, and defines a custom drawer content component (CustomDrawerContent) that renders the app logo, navigation links, and a settings modal.
